# HRNet

Comprehensive and modularized implementation of [HRNet](https://arxiv.org/abs/1908.07919).

## Installation

Dependencies:

* Git
* Python 3.10
* Poetry

```shell
git clone https://github.com/laclouis5/HRNet.git
cd HRNet
poetry install
```

## Show the Network Summary

```shell
poetry run python hrnet.py
```
